## Getting Started

This project is built with **Astro** and managed with **Bun**.

### Development

Install dependencies:

```bash
bun install
```

Start the development server:

```bash
bun run dev
```

### Build

To generate a static build:

```bash
bun run build
```

The output will be in the `dist/` directory.
